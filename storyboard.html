Project Overview

This storyboard outlines the visual structure, layout, and user flow of my portfolio website, including how the navigation works, the design theme, and specific page functionalities.

Layout Design (Wireframe)

The wireframe includes a structured layout with navigation, header, content areas, and footer for a smooth user experience.

Web Pages and Their Purpose

  • Home Page: Introduces the portfolio and provides an overview of my background.
  • Background Page: Contains information about likes, dislikes, hobbies, educational journey, and personal background.
  • Professional Page: Highlights career goals, the vision, mission, skills, qualifications, and job requirements to become a Software Developer.
  • Personality Page: Provides insights into my strongest personality trait and hobby and how they impact my academic and personal life.
  • Personal Development Page: Provides a brief summary of Chapters 4-7 of "Twelve Pillars" and a reflection of how I can apply the lessons learnt to my life. Includes the skills that I have developed so far and the degree I am currently pursuing.

Site Theme

The theme of the site is vibrant and professional, targeting young professionals and students as well as organizations with job vacancies in the Computer Science field. It features a color palette of various shades of purple, conveying creativity and professionalism. The design includes personal images and graphics to enhance engagement, along with subtle icons for navigation. This approach aims to create an inviting atmosphere that reflects my personality and academic aspirations, making the portfolio memorable and user-friendly.

Features

The following features are intended to be included on the website:

  • Responsive Navigation Bar: The navigation bar will be styled with CSS for a clean, horizontally aligned layout, ensuring user-friendly navigation across the site.
  • Clickable Profile Picture: The profile picture will link directly to an email prompt, allowing users to contact me quickly by clicking on the image.
  • Visually Structured Section Containers: Each section will be presented in its own container with an image, title, and description, enhancing organization and visual appeal.
  • Responsive Image and Card Layout: Card-based layouts will ensure that content is well-organized and adapts to various screen sizes while maintaining aesthetic balance.
  • Images: Images will be included to enhance visual appeal, communicate ideas quickly, and engage users more effectively, breaking up large blocks of text.
